# Catalog IQ: Walmart Global Tech Prototype

This is a high-fidelity **"Proof of Work"** prototype developed by **Saurabh Chawda** for the Senior Product Manager role at **Walmart Tech India**.

## 🚀 Vision
To build a world where the retail catalog moves at the speed of thought, eliminating the engineering bottleneck through autonomous governance and executive-level risk guardrails.

## 🛠️ Tech Stack
- **Framework:** React 18 + Vite (Vike V1 Design)
- **Styling:** Tailwind CSS v4 (Dark Theme: Walmart Cyber-Tech)
- **State Management:** Zustand (Executive Resilience Engine)
- **Icons:** Lucide-React (High-Density Enterprise Set)

## 📦 Key Modules
1.  **Strategic ROI Dashboard:** Quantifies the business value ($185k/qtr ROI).
2.  **Taxonomy Engine:** Deep hierarchy builder with AI Co-Pilot & Blast Radius Analysis.
3.  **Ingestion Portal:** Real-time validation with "Round-Trip" Excel Error Exports.

## 🏃 Quick Start
```powershell
cd item-catalog-simulator
npm install
npm run dev
```
Open `http://localhost:3000` to begin the executive demo.
